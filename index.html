⚙ Настройки ИИ

Как получить бесплатный ключ (один раз, 3 минуты):
1. Зайдите на huggingface.co/join и зарегистрируйтесь.
2. Подтвердите почту.
3. Откройте Settings → Access Tokens.
4. Create new token → тип Read → Create.
5. Скопируйте ключ (hf_...) и вставьте в поле выше. Он сохранится только в вашем телефоне.

Бесплатного лимита хватает на несколько генераций в месяц; для безлимита подключим платный ключ позже.
